# A-real-time-sign-language-recognition-system-
A real time sign language recognition system configured to operate on low compute hardware without graphical processing acceleration 
Developed a real-time Sign Language Recognition System using MediaPipe for hand-landmark detection and a Random Forest classifier for gesture prediction. Achieved 93.4% accuracy and 0.08s inference time, enabling smooth CPU-based operation without GPU. The system recognizes A–Z alphabets and digits 0–9 and convert them into text and speech, offering a low-cost accessibility solution.
